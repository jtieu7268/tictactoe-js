# tictactoe-js
An implementation of 2-player Tic Tac Toe.

Press 'START' to play. Enter the player names and submit. Each player takes a turn placing their marker on the board by clicking on the desired position until one player wins or there is a tie.

## Usage

To play, click [here](https://jtieu7268.github.io/tictactoe-js/).

## Technologies

Implemented using Javascript, HTML, and CSS. Deployed using GitHub Pages.
